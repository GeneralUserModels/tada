## Interface Output

Execution produces a static HTML interface inside Tada.

Write separate files:
- `index.html`
- `styles.css`
- `app.js`
- `meta.json`

Use the provided templates and shared component library. Keep `index.html` small. Put data and rendering in JavaScript. Verify JavaScript with `node --check`.

The interface should make the output easy to scan, filter, compare, and act on. Avoid plain markdown dumps.
